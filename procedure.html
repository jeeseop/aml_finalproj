Procedure to Reproduce

Prerequisites

  • Computer: Jetson TX2

  • Python version: 3.6.8 (used python virtual environment)

  • pillow version: 6.2.1

  • scipy version: 1.2.2

  • numpy version: 1.16.1

  • keras version: 2.3.1

  • h5py version: 2.9.0

  • tensorflow-gpu version: 1.14.0+nv19.10

System Setup

  • Install tensorflow-gpu in Jetson TX2 with python virtual environment:

    https://docs.nvidia.com/deeplearning/frameworks/install-tf-jetson-platform/index.html

  • Install the required packages described above with same version

Run the Code

  • Clone the code from repo[3] (original code from repo[1], modified by Jeeseop(myself))

  • execute: $ python3 tran_models.py

    - This portion will train the MNIST, and CIFAR-10 with and without Defensive distillation [5]

    - I used distillation temperature: 0, 20, 40, 60, 80, 100 to make trained data

    - Generated data files can be used in test_attack.py

  • execute: $ python3 test_attack.py

    - This portion will show the

    --- original label of the input,

    --- target label that attack targeted,

    --- classification result with attacked input, and

    -- distortion made by attack

    - Attack is successful if the classification result shows the same target label that attack targeted

    - Attack is useful if the distortion value is regulated as small value in every case
